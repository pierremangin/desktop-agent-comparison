2. Finalités

Ce site a pour finalité principale de publier un contenu éditorial et comparatif sur des outils logiciels.

Les données techniques éventuellement traitées le sont pour :

  • assurer l’affichage et le bon fonctionnement du site ;
  • assurer la sécurité du site et prévenir les abus ;
  • héberger et servir les contenus, images, polices et données nécessaires à la consultation ;
  • le cas échéant, établir des statistiques techniques ou de fréquentation strictement nécessaires à l’exploitation du site.

3. Données traitées

Lors de votre navigation, les catégories de données suivantes peuvent être traitées :

  • adresse IP ;
  • données techniques de connexion et de navigation ;
  • date, heure, requêtes HTTP, informations de navigateur et de terminal ;
  • journaux techniques et de sécurité générés par l’hébergeur ou les prestataires techniques ;
  • le cas échéant, traceurs strictement nécessaires au fonctionnement du site.

Le site n’a pas vocation à proposer un compte utilisateur ni à collecter des données sensibles.

4. Base légale

Les traitements décrits sur cette page reposent, selon les cas, sur :

  • l’intérêt légitime du responsable du traitement pour exploiter, sécuriser et améliorer le site ;
  • le respect d’obligations légales ;
  • le consentement de l’utilisateur, lorsqu’il est légalement requis.

5. Destinataires

Les données peuvent être accessibles, dans la limite de leurs besoins respectifs, aux prestataires techniques utilisés pour faire fonctionner le site, notamment :

  • l’hébergeur du site ;
  • le prestataire de base de données / backend ;
  • les fournisseurs de bibliothèques ou ressources chargées côté navigateur, le cas échéant.

Dans la configuration actuelle du site, cela peut inclure notamment OVHcloud, Supabase, jsDelivr et Google Fonts.

Certains prestataires techniques ou certaines ressources tierces peuvent entraîner des transferts de données techniques, notamment d’adresse IP, hors de l’Union européenne.

Lorsque de tels transferts existent, ils sont encadrés par les mécanismes juridiques applicables prévus par la réglementation.

Les données de navigation et journaux techniques sont conservés pendant une durée limitée, proportionnée à leur finalité technique, de sécurité ou d’exploitation, puis supprimés ou anonymisés selon les pratiques des prestataires concernés.
